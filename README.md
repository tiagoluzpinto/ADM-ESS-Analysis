# Multivariate Data Analysis - European Social Survey (Round 10) 📊

This project was developed for the **ADM** course at **ISCTE-IUL** (2025-2026). The goal is to analyze social attitudes and values in Montenegro using the European Social Survey (ESS) data from 2020-2022.

## 🎯 Objectives
- Data cleaning and preparation.
- Principal Component Analysis (PCA) to define dimensions of social values.
- Cluster Analysis to identify patterns among respondents.
- Socio-demographic characterization of the found groups.

## 🛠️ Tools
- **Language:** R
- **Environment:** RStudio
